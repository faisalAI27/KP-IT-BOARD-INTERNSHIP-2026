## Day 2: Artificial Neural Networks and GPU Computing

* Studied the basic concepts of Artificial Neural Networks (ANNs).
* Learned about neurons, weights, biases, input layers, hidden layers, and output layers.
* Understood the role of activation functions in adding non-linearity to neural networks.
* Explored forward propagation and how input data passes through the network.
* Learned about cross-entropy loss and how it is used in multi-class classification.
* Studied the chain rule and its importance in backpropagation.
* Understood partial derivatives and how they help calculate gradients for each weight and bias.
* Learned how gradients are used to update model parameters during training.
* Explored GPU computation and how it speeds up deep learning operations.
* Understood the difference between CPU threads and GPU parallel processing.
* Learned how PyTorch connects with GPU backends for faster computation.
* Studied the role of CUDA in enabling NVIDIA GPU computation.
* Learned about cuDNN and how it provides optimized deep learning operations on NVIDIA GPUs.
* Understood that PyTorch uses lower-level C++/CUDA backends for efficient tensor and neural network computations.

**Purpose:** To understand how neural networks learn and how GPU acceleration improves deep learning computation.
